## Getting started

## Usage

```dart 
   VerificationCode(
      height: 50,
      style: CodeStyle.form,
      maxLength: 4,
      itemWidth: 50,
      onCompleted: (String value) {
          print("CodeStyle.form value=$value");
      },
    ),
```
